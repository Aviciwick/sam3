### 部署计划

1.  **创建隔离的 Conda 环境**
    *   创建名为 `sam3` 的环境，指定 Python 3.12。
    *   此操作在用户私有目录进行，不影响系统全局配置。

2.  **配置环境内 CUDA 工具包**
    *   在 conda 环境中安装 `cuda-toolkit=12.6`（来自 `nvidia` 通道）。
    *   这将提供环境专用的 `nvcc` 编译器，满足项目对 CUDA 12.6+ 的要求，同时不修改系统 `LD_LIBRARY_PATH`，利用 conda 的激活脚本自动管理路径。

3.  **安装 PyTorch 及依赖**
    *   安装 `torch==2.7.0` (配合 CUDA 12.6)，遵循项目 README。
    *   安装 `sam3` 项目及其依赖（使用 `pip install -e .`）。

4.  **验证环境与隔离性**
    *   **步骤 1 (环境内)**: 激活 `sam3`，执行 `nvcc -V`，预期显示 **12.6**。
    *   **步骤 2 (功能验证)**: 在环境内运行 Python 脚本，检查 `torch.cuda.is_available()` 为 `True`，并打印 PyTorch 版本。
    *   **步骤 3 (环境外)**: 退出环境，执行 `nvcc -V`，预期显示系统原有版本（检测到为您当前的 11.5），证明未污染系统环境。
